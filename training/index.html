Web Applications Development

This course teaches the fundamentals of web app development from beginner to advanced level using the following popular web frameworks:

  • Django Python Web Framework
  • Laravel PHP Web Framework
  • MEAN JavaScript Stack

Machine Learning and Data Science/Visualization

This course teaches the fundamentals of data science and machine learning from beginner to advanced topics using the following popular tools and frameworks:

  • R
  • NumPy and Scikit-Learn Python Libraries
  • D3.js and other JavaScript Visualization Libraries
  • Tensorflow
  • Apache Spark and the Hadoop Ecosystem

Ethical Hacking and Computer Forensics

This course introduces the learner to the fundamentals of Cyber-Security and the different ways of protecting computing systems against cyber threats and malicious attacks. The student will gain the following:

  • Ethical Hacking skills using Python
  • Penetration Testing skills using Kali Linux
  • Computer Forensics skills and tools
